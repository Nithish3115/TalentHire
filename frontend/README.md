# Job Fair Portal MVP - Frontend

This is the React frontend for the Job Fair Portal MVP.

## Tech Stack
- React 18
- React Router DOM
- Tailwind CSS
- Axios for API calls

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   The application will run on \`http://localhost:5173\` by default with Vite.

## Backend Dependency
Ensure the Spring Boot backend is running on \`http://localhost:8080\` before interacting with API-driven features.
